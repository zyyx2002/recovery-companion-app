## 🏗️ 项目架构

### 技术架构图
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   移动端APP     │    │   后端API服务   │    │   管理后台      │
│  (React Native) │◄──►│   (Node.js)     │◄──►│   (Next.js)     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         │                       │                       │
         ▼                       ▼                       ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   本地存储      │    │   PostgreSQL    │    │   数据统计      │
│  (AsyncStorage) │    │   (Neon)        │    │   (Recharts)    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 核心组件
- **移动端**: React Native + Expo + Zustand
- **后端**: Node.js + Express + Drizzle ORM
- **数据库**: PostgreSQL (Neon)
- **管理后台**: Next.js + Tailwind CSS
- **认证**: JWT + bcrypt
- **推送**: Expo Notifications
